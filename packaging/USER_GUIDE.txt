JAV一网打尽 — Windows 便携版使用说明
========================================

【系统要求】
  - Windows 10 / 11（64 位）
  - 无需安装 Python，解压即可使用
  - 建议使用 Chrome、Microsoft Edge 或 115 浏览器

【文件夹说明】
  JAV一网打尽.exe    桌面程序（已内置运行环境与依赖）
  extension\         浏览器扩展（必须与 exe 放在同一目录）
  Start.bat          可选：双击启动桌面程序
  USER_GUIDE.txt     本文件

【隐私说明 — 发给他人前必读】
  - 本 ZIP 不含您的库路径、UNC 地址或用户名
  - 对方首次运行会在自己的 %USERPROFILE%\.jav-manager\ 生成全新配置
  - 请勿把源码目录、.jav-manager 文件夹或项目里的 txt 一并打包给别人

【第一次使用 — 桌面端】
  1. 解压整个文件夹到任意位置
  2. 双击 JAV一网打尽.exe 或 Start.bat
  3. 若 Windows 提示「未知发布者」，点「更多信息」→「仍要运行」

【第一次使用 — 浏览器扩展】
  1. 保持桌面程序正在运行
  2. Chrome / Edge 打开 chrome://extensions/ 或 edge://extensions/
  3. 开启「开发者模式」→「加载已解压的扩展程序」
  4. 选择本文件夹内的 extension 目录
  5. 打开 javdb.com；桌面程序会弹出「是否允许扩展连接」，点「是」
  6. 扩展弹窗应显示「已连接」（无需复制配对码）

【数据保存在哪】
  配置与操作数据库：%USERPROFILE%\.jav-manager\
  影片文件仍在您自己设置的库目录中，不会被打包进程序。

【更新版本】
  覆盖 JAV一网打尽.exe 和 extension 文件夹，并在浏览器扩展页点「重新加载」。

【版本】
  见程序标题栏「JAV一网打尽 vX.X.X」
