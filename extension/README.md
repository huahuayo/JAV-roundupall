# JAV Manager Bridge 扩展安装说明

本扩展用于 **Microsoft Edge** 与 **115 浏览器**，将当前标签页 URL / 标题实时同步到 JAV Manager 桌面程序。

Edge 与 115 需**分别安装一次**（加载同一个 `extension` 文件夹即可）。

## 1. 启动桌面程序

运行 `JAV Manager.exe` 或 `run.bat`，在 **浏览器监控** 面板点击 **复制配对码**。

## 2. Microsoft Edge

1. 打开 `edge://extensions/`
2. 开启左下角 **开发人员模式**
3. 点击 **加载解压缩的扩展**
4. 选择文件夹：`jav-manager\extension`（或 `dist\extension`）
5. 点击扩展图标 → **配置配对码 / 端口** → 粘贴配对码 → 保存

## 3. 115 浏览器

1. 打开 115 浏览器的扩展管理页（一般在 设置 → 扩展程序）
2. 开启 **开发者模式** / **加载扩展**
3. 选择同一文件夹：`jav-manager\extension`
4. 在扩展选项页粘贴配对码并保存

> 115 基于 Chromium，安装步骤与 Edge 类似；若菜单名称不同，找「加载已解压的扩展程序」即可。

## 4. 验证连接

- 扩展弹窗显示 **已连接**
- 桌面程序 **浏览器监控** 面板显示 `Edge: ● 已连接` 或 `115: ● 已连接`
- 切换网页时，当前页面 URL 会实时更新

## 5. JavDB 列表页布局（v1.1）

在 **JavDB 列表页**（首页、搜索、分类等）右上角会出现 **⚙ 设置** 按钮：

- **满屏显示**：开启后容器铺满浏览器宽度，消除左右大段空白
- **每行贴纸数量**：2–12，预览卡片（贴纸）自动缩放排布

> 仅对含 `#videos` 网格的列表页生效；点击进入 **`/v/...` 详情页** 时不会改动原布局。

设置会保存在扩展本地存储中，刷新列表页后仍然有效。

## 通信说明

- 协议：WebSocket `ws://127.0.0.1:17892`（仅本机）
- 认证：首次连接发送 `hello` + 配对码
- 消息：`tab_update` 推送当前活动标签页

后续功能（一键收藏、页面元数据抓取等）将通过同一通道扩展。
